Steps for tests running:
1. Open solution LoginFormTests.sln in Visual Studio;
2. Right click on Tests folder in Solution Explorer and click "Run Unit Tests".
3. If you need to run only some certain test, you can choose it in ReSharper unit test explorer (ReSharper -> Unit test -> Unit test (CTRL + ALT + T)) and run it by Ryght click from context menu.
4. If you need to check faults logging, you can edit some variable value in TestData class, which discribes expected data. Log path location is "~\AppData\Roaming\LoginFormTests".
